Setting Up the Predix Developer Environment

Set Predix platform user accounts, install Cloud Foundry CLI client, and configure proxy settings.

  1. Register for a Predix platform user account. See Registering for a Predix Platform User Account.
  2. Install the latest stable binary version of the Cloud Foundry CLI client. See https://github.com/cloudfoundry/cli#downloads.
  3. Configure your proxy settings to access the Internet. See Defining Proxy Connections to Remote Resources.
