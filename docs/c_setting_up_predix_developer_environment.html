Creating the Predix Developer Environment

Complete the following tasks to create the Predix developer environment.

  1. Register for a Predix platform user account. See Registering for a Predix Platform User Account for more information.
  2. Install the latest version of the Git client for your operating system. See https://git-scm.com/downloads for more information.
  3. Create a GitHub account. See https://github.com/join for more information.
  4. Add a Secure Shell (SSH) key to your GitHub account. See https://help.github.com/articles/generating-an-ssh-key/ for more information.
  5. Verify that you have access to the PredixDev organization after you log into your GitHub account.
  6. Install the latest stable binary version of Cloud Foundry CLI . See https://github.com/cloudfoundry/cli#downloads for more information.
  7. Configure your proxy settings to access the Internet. See Defining Proxy Connections to Remote Resources.
