Script REST API

Use the Script REST API to create, delete, and retrieve a script or a list of scripts.

Method Path Content Type Description
PUT /system/scripts/:id application/javascript Creates a script.
DELETE /system/scripts/:id   Deletes a script.
GET /system/scripts/:id   Retrieves a script.
GET /system/scripts   Retrieves a list of scripts.
