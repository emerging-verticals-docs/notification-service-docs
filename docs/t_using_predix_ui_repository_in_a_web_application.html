Using a Predix UI Repository in a Web Application

Set up the bower task runner utility to import the correct version of each Predix UI repository required for a web application.

  • Set up the required services for Predix UI. See Setting Up the Predix UI Developer Environment for more information.
  • Name of your web application <web_app>
  • Locate the project directory of your web application <web_app_project> where the following files will be installed:
    • webcomponents-lite polyfill
    • Predix UI repository
Note: For more information, see https://predixdev.github.io/predix-ui/?show=getting_started&type=local.

Each Predix UI repository includes a bower.json file that lists other required repositories. This is displayed as a series of @import declarations in the respective file. For more information, see Application Development with Predix UI for more information.

  1. Change to the project directory of your web application <web_app_project>:
    cd <web_app_project>
  2. Install the webcomponents-lite polyfill into your project directory by using the bower package-management utility:
    bower install webcomponentsjs --save
  3. Add the webcomponents-lite polyfill to your web application as the first line in the <head> section of the HTML page . For example
    <!DOCTYPE html>

    <html>

        <head>
            <script src="bower_components/webcomponentsjs/webcomponents.js"></script>

        </head>

     </html>
  4. Install the Predix UI repository as a web component into the project directory by using the bower package-management utility:
    bower install https://github.com/PredixDev/<px-repository> --save
  5. Import the Predix UI repository as a web component into your web application by adding it to the <head> section of the HTML page. For example:
    <!DOCTYPE html>

    <html>

        <head>

            <script src="bower_components/webcomponentsjs/webcomponents.js"></script>

  
            <link rel="import" href="/bower_components/<px-repository>/<px-repository>.html"/>

        </head>

     </html>

    The Predix UI repository files are installed in the bower_components directory.

  6. Call the Predix UI repository in your web application by adding it to the <body> section of the HTML page. For example:
    <!DOCTYPE html>

    <html>
    
      <head>

        <script src="bower_components/webcomponentsjs/webcomponents.js"></script>

  
        <link rel="import" href="/bower_components/<px-repository>/<px-repository>.html"/>


     </head>

     <body>

         <px-repository></px-repository>

     </body>


  </html>
