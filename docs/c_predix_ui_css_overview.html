Predix UI CSS Overview

The Predix UI platform includes the Predix UI CSS libraries in the PredixDev public GitHub organization to create intricate user interactions in a web application.

As a developer. you can consume as many Predix UI CSS libraries as required to configure the user interface to your specific application requirements. Predix UI CSS also includes Predix UI collections organized by common functions (such as lists or buttons) for convenient use when configuring a web application. The modular architecture includes defined variables and default variants that provide the flexibility to change and configure existing web applications, rather than use a large, complex, single-component module.

Note: For more information, see https://predixdev.github.io/predix-ui/?show=use_css&type=local.
The Predix UI CSS libraries are organized into the following categories:
  • Visual
  • Layout
  • Utilities
