Predix UI CSS Overview

The Predix UI platform includes the Predix UI CSS modules in the PredixDev public GitHub organization that establish the base typography, layout, and content design in a web application.

As a developer. you can consume as many Predix UI CSS modules (https://www.predix-ui.com/) as required to configure the user interface to your specific application requirements. Predix UI CSS also includes collections organized by common functions (such as lists or buttons) for convenient use when configuring a web application. The modular architecture includes defined variables and default variants that provide the flexibility to change and configure existing web applications, rather than use a large, complex, single-component module.

Note: For more information, see https://predixdev.github.io/predix-ui/?show=use_css&type=local.
The Predix UI CSS modules are grouped into the following libraries:
  • Predix UI CSS Visual Library
  • Predix UI CSS Layout Library
  • Predix UI CSS Utilities Library
