Associating a Card with a Deck

Procedure for associating a card with a deck in the View service database

Select a REST client (such as Postman) that can execute API requests to use the View service.

For more information about this API, see the API Documentation.

Associate one or more cards to a deck in the View service database and retrieve all cards.
  1. Associate a card with a deck once they are created in the View service database. For example:
    1. Enter the following POST request:
      [POST] /api/decks/1/cards/add
    2. Copy the following JSON block into the request body:
      ["1", "2"]
  2. Retrieve all cards in a deck from the database in JSON format. For example:
    1. Enter the following GET request:
      [GET] /api/decks/1/cards
    2. Copy the following JSON blocks into the request body:
      {
    "id": 1,
    "title": "Contra Costa Deck",
    "cards":[
        {
            "id": 1,
            "title": "San Ramon Temperature",
            "slug": "my-temperature-card",
            "attributes": {
                "postal": "94582",
                "unit": "F"
            }
        },
        {
            "id": 2,
            "title": "Walnut Creek Temperature",
            "slug": "my-temperature-card",
            "attributes": {
                "postal": "94595",
                "unit": "F"
            }
        }
    ]
}
