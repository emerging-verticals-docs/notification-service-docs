Troubleshooting Predix UI Seed Deployment

You may encounter this issue that when deploying the Predix UI Seed to Cloud Foundry.

Failure message is displayed

You may receive a FAILED error message when you fail to deploy the Predix UI Seed to Cloud foundry.

Cause

The manifest.yml file is not configured properly.

Solution

See Deploying the Predix UI Seed to Cloud Foundry for information on configuring the manifest.yml file.
