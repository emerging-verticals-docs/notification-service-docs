Predix UI Seed Setup

Configure the required tools to install and deploy the Predix UI Seed to Cloud Foundry.

As a developer, you do not need any Predix service to deploy the Predix UI Seed to your local development environment or Cloud Foundry.

Task Roadmap

# Task Information
1 Set up the Predix developer environment. See Setting Up the Predix Developer Environment for information.
2 Install the latest compatible version of the Node.js runtime environment, allowing you to then set up the developer tools needed to install the Predix UI Seed. See https://nodejs.org/en/download/ for information..
3 Modify the file permissions for the Node.js npm package-management utility. See https://docs.npmjs.com/getting-started/fixing-npm-permissions for information.
4 Install the Predix UI Seed files. See Installing the Predix UI Seed for information.
5 Deploy your Predix UI Seed web application to Cloud Foundry. See Deploying the Predix UI Seed to Cloud Foundry for information.
