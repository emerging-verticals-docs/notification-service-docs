Delete Configuration

This service deletes a configuration.

METHOD DELETE
URI DELETE tenants/{tenant_uuid}/configuration/{uuid}

Path Parameters

Parameter Description Required?
tenant_uuid UUID of the tenant. Yes
uuid UUID of the configuration. Yes
