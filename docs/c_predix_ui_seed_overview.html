Predix UI Seed Overview

The Predix UI Seed uses the Predix UI platform to deploy to your local development environment or Cloud Foundry.

The Predix UI Seed is a web application starter kit hat demonstrates the use of the Predix UI platform to implement the following features:
  • Application Navigation
  • Responsive Layout
  • Routing
  • Data Binding
  • View Switching
  • Data Visualization
  • Date and Time Input
  • Service Integration

The Predix UI Seed does not depend on any Predix service to deploy to your local development environment or Cloud Foundry.

As a developer, you can copy, extend, and deploy the Predix UI Seed to suit your specific application requirements. For more information, see https://predixdev.github.io/predix-ui/?show=use_seed&type=local.
