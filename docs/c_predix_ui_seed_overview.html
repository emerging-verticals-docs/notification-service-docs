Predix UI Seed Overview

The Predix UI Seed web application starter kit uses the Predix UI platform to deploy to your local development environment or Cloud Foundry.

The Predix UI Seed does not depend on any Predix service to deploy to your local development environment or Cloud Foundry.

As a developer, you can copy, extend, and deploy the Predix UI Seed to suit your specific application requirements. For more information, see https://predixdev.github.io/predix-ui/?show=use_seed&type=local.
