Getting Started with View Service

Set up a UAA service instance as the trusted issuer. See UAA Service Setup for more information.

Follow these steps to set up the View service:
  1. Create a View service instance. See Creating a View Service Instance.
  2. Bind the View service instance to an application. See Binding an Application to a View Service Instance.
  3. Create an OAuth2 Client for the View service. See Create an OAuth2 Application for Your Application.
  4. Update the OAuth2 client to set up user authentication and access to the View service. See Using UAAC to Update the Oauth2 Client to Use View Service.
