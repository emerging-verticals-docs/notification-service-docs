Update Configuration

This service updates the configuration.

METHOD PUT
URI PUT tenants/{tenant_uuid}/configurations/{uuid}
Sample Request 
{
    "protocol": "smtp",
  	"host": "smtp.gmail.com",
  	"port": 587,
  	"smtpAuth": "true",
  	"smtpStarttlsEnable": "true",
  	"mailFrom": "testge.testge@gmail.com",
  	"mailUsername": "testge.testge",
  	"mailPassword": "getest123",
  	"mailReturnPath": null
}

Path Parameters

Parameter Description Required?
tenant_uuid UUID of the tenant. Yes
uuid UUID of the configuration. Yes

Request Body JSON Parameters

Note: Configu details will be over-ridden.
Parameter Description Required?
protocol Protocol of the mail server. Optional
host Host of the mail server. Optional
smtpAuth SMTP auth for the mail server. True or false value. Optional
smtpStarttlsEnable SMTP starttls enable for the mail server. True or false value. Optional
mailForm From email address. Optional
mailUsername Mail server username for authentication. Optional
mailPassword Email server password for authentication. Optional
mailReturnPath Mail return path. Optional
