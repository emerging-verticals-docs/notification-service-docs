Get Configuration by UUID

This service returns a uuid.

METHOD GET
URI GET /tenants/{tenant_uuid}/configurations/{uuid}

Path Parameters

Parameter Description Required?
tenant_uuid UUID of the tenant. Yes
uuid UUID of the configuration. Yes
