Add Configuration

This service creates or adds configurations.

METHOD POST
URI POST /tenants/{tenant_uuid}/configurations
Sample Request 
[
            {
      "protocol": "smtp",
      "host": "smtp.sparkpostmail.com",
      "port": "587",
      "smtpAuth": "true",
              "smtpStarttlsEnable": "true",
              "mailFrom": "clarity-notifications@engagement.digital.ge.com",
              "mailUsername": "SMTP_Injection",
              "mailPassword": "password",
              "mailReturnPath": null
    }
]

Path Parameters

Parameter Description Required?
tenant_uuid UUID of the tenant. Yes

Request Body JSON Parameters

Parameter Description Required?
protocol Protocol of the mail server. Yes
host Host of the mail server. Yes
smtpAuth SMTP auth for the mail server. True or false value. Yes
smtpStarttlsEnable SMTP starttls enable for the mail server. True or false value. Yes
mailForm From email address. Yes
mailUsername Mail server username for authentication. Yes
mailPassword Email server password for authentication. Yes
mailReturnPath Mail return path. Optional
