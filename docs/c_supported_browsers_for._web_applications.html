Supported Browsers for Web Applications

The Predix UI platform has been tested to support these web broswers and opeating systems.

The following minimum web browser versions have been fully tested to support all Predix UI repositories that are available in the PredixDev public GitHub organziation..

Browser OS
Chrome 48.0
  • Windows 8.1
  • MacOS 10.10
Firefox 43.0
  • Windows 8.1
  • MacOS 10.10
Internet Explorer 10 Windows 7
Internet Explorer 11 Windows 8.1
Safari 9.0 MacOS 10.10
Note: File a GitHub issue if you encounter a problem using a Predix UI repository with a supported web browser. For more information on filing a GitHub issue, see https://guides.github.com/features/issues/.
