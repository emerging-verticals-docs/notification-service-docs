Using a Predix UI CSS Library in a Web Application

The Predix UI CSS libraries ensure consistent design in a web applicaion that adapts the presentation to different devices.

Refer to the API reference documentation for each Predix UI CSS library in the Predix UI library (http://predixdev.github.io/predix-ui/) for specific configuration information regarding:
  • Feature flags
  • Style variables
  • Import order
Note: See https://predixdev.github.io/predix-ui/?show=use_css&type=local for more information on using a Predix UI CSS library in the PredixDev public GitHub organization.

A cascading style sheet (CSS) contains style rules that are applied to elements in a HTML page. CSS styles define how HTML elements are displayed and where they are positioned on the page.

Follow these steps to use a Predix UI CSS library in your web application (px-buttons-design is used as an example):

  1. Change to the project directory of your web application <app_proj>:
    cd <app_proj>
  2. In the project SASS file<app_proj>.scss, define the following order for importing the default Predix UI CSS library:
    // Settings
   @import "px-colors-design/_settings.colors.scss";

   // Tools

   // Generic
   @import "px-normalize-design/_generic.normalize.scss";
   @import "px-box-sizing-design/_generic.box-sizing.scss";
   @import "px-helpers-design/_generic.helpers.scss";

   // Base
   @import "px-flexbox-design/_base.flexbox.scss";
   @import "px-viewport-design/_base.viewport.scss";
   @import "px-typography-design/_base.typography.scss";

   // Meta

   // Objects

   // Component
   // Your css goes in here

   // Trumps
   @import "inuit-clearfix/_trumps.clearfix.scss";
   @import "px-spacing-responsive-design/_trumps.spacing-responsive.scss";
   @import "px-widths-responsive-design/_trumps.widths-responsive.scss";
  3. Install the px-buttons-design library by using the bower package-management utility:
    bower install --save px-buttons-design
  4. In the project SASS file, import the px-buttons-design library he :
    // Settings
   @import "px-colors-design/_settings.colors.scss";

   // Tools

   // Generic
   @import "px-normalize-design/_generic.normalize.scss";
   @import "px-box-sizing-design/_generic.box-sizing.scss";
   @import "px-helpers-design/_generic.helpers.scss";

   // Base
   @import "px-flexbox-design/_base.flexbox.scss";
   @import "px-viewport-design/_base.viewport.scss";
   @import "px-typography-design/_base.typography.scss";

   // Meta

   //Object
   @import "px-buttons-design/_objects.buttons.scss";


   // Component
   // Your css goes in here

   // Trumps
   @import "inuit-clearfix/_trumps.clearfix.scss";
   @import "px-spacing-responsive-design/_trumps.spacing-responsive.scss";
   @import "px-widths-responsive-design/_trumps.widths-responsive.scss";
  5. Enable px-buttons-design library features. The following example enables the $inuit-enable-btn--small feature by setting the feature flag to true before the px-buttons-design library @import statement. 
    //Object

$inuit-enable-btn--small : true;
@import "px-buttons-design/_objects.buttons.scss";
  6. Change a Predix UI CSS library style variable (if available) by setting a new value and inserting the style variable before the library @import statement:
    //Object

$inuit-btn-color : $black;
@import "px-buttons-design/_objects.buttons.scss";
  7. In your web application, add the appropriate px-buttons-design call to the desired element.
    <!DOCTYPE html>
    <html>
      <head>
     </head>
     <body>
         <button class="btn btn--small">Small</button>
     </body>
  </html>

    For more information, see the px-buttons-design library API reference.
