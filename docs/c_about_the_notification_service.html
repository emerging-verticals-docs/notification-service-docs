About the Notification Service

The Predix Notification service is a notification service that notifies list of recipients through email. The service provides a simple and scalable way to provide notification services for user applications. The Predix Notification service is available as a service tile in the Predix Cloud and allows users to create, update, and delete tenant and related configurations through a REST interface.
Note: Consumers of the notification service will need to come up with their own SMTP provider and configure it during service provisioning. The configuration is a one-time setup and multiple configurations can be maintained for each tenant.
