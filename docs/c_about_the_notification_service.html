About the Notification Service

The Predix Notification service is a notification service that notifies list of recipients via email. The service provides a simple and scalable way to provide notification services for user applications. The Predix Notification service is available as a service tile in the Predix Cloud and allows users to create, update, and delete tenant and related configurations through a REST interface.
Important: Data at REST is not encrypted. Use discretion when storing sensitive data.
