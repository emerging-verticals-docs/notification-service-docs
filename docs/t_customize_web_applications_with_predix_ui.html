Customizing Web Applications with Predix UI

You can use the Predix UI platform to customize web applications that run on top of Predix services and data.

The Predix UI platform (http://predixdev.github.io/predix-ui/) includes these elements to extend and customize a web application:
  • A set of Predix UI repositories to add features or access services following consistent UX practices, standards, and specifications.
  • A set of Predix UI CSS modules that establish consistent UI design.
Table 1. Web Application Customization Options
Option Description
Use the Predix UI repositories in the PredixDev public GitHub organization for reusable user interface components (such as application navigation and context browsing). See Using a Predix UI Repository in a Web Application.
Use the Predix UI CSS modules in the PredixDev public GitHub organization to satisfy custom requirements to create more intricate user interactions. See Installing the Predix UI CSS Starter Kit
Build a custom Predix UI component that suits your specific application requirements. . See Build custom components with the generator.
Use the View service to define relationships between card and deck components. See About View Service.
