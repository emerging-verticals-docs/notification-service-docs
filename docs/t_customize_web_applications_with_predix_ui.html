Customizing Web Applications with Predix UI

Customize web applications tasks

As a developer, you can use the Predix UI platform to customize web applications that run on top of Predix services and data.
Table 1. Web Application Customization Options
Option Description
Use the Predix UI repositories in the PredixDev public GitHub organization for reusable user interface components (such as application navigation and context browsing). See Using a Predix UI Repository in a Web Application.
Use the Predix UI CSS libraries in the PredixDev public GitHub organization to satisfy custom requirements to create more intricate user interactions. See Configuring the Predix UI CSS Starter Kit
Build a custom user interface component that suits your specific application requirements. . See https://predixdev.github.io/predix-ui/?show=use_component_generator&type=local.
