Predix UI Templates Repositories

The Predix UI templates repositories can lay out the entire user interface (or region of the user interface) of a web application.

Predix UI includes a set of templates repositories in the PredixDev public GitHub organization. The Predix UI templates repositories focuse on the content structure or layout, not the actual content. when creating the user interface of a web application.

As a developer, you can use the code in the following Predix UI templates repositories to produce these interactions.

Table 1. Predix UI Templates Repositories
Repository Description Documentation
px-card User interface component that can layout a region of a web application. API Reference
px-dashboard User interface component that renders a deck with a given URL in a web browser. API Reference
px-deck User interface component that contains a collection of card objects. API Reference
px-deck-selector User interface component that displays a dropdown of deck objects and returns the URL for the selected deck. API Reference
px-kpi A data visualization component that displays key performance indicators (KPI) in a dashboard layout. API Reference
px-login A user interface component that displays login/logout button. API Reference
px-sample-cards Examples which demonstrate how to make cards objects. API Reference
px-widget-cards Card objects consisting of a header area and content area. API Reference
Note:

The API reference documentation for individual repositories also includes usage instructions and examples.

File a GitHub issue if you encounter a problem using a Predix UI repository. For more information on filing a GitHub issue, see https://guides.github.com/features/issues/.
