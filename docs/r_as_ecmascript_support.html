ECMAScript Support

The scripting engine provides support for the ECMAScript language.

The ECMAScript language support includes the following JavaScript components:
  • Generators
  • Arrow functions
  • Binary and octal literals
  • let variables
  • String methods:
    • .contains() (replaced by .includes() in the ES6 specification)
    • .startsWith()
    • .endsWith()
    • .codePointAt()
    • .repeat()
    • .normalize()
    • String.fromCodePoint
See the JavaScript Reference for details.
