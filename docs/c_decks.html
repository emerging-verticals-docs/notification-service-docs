Decks

A deck is a component that uses a specified layout to determine the arrangement of cards. You can use a View service API to save the complete context of a deck. Saving a card within a deck calls the same function that a card uses to save its own state. See Customizing Web Applications with Predix UI for information on how cards and decks are used.

For more information about this API, see the API Documentation.

The ability to see, modify, and delete decks is governed by access-control rules associated with the UAA services. Access to these objects can be restricted by user, role, or group.
