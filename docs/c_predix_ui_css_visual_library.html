Predix UI CSS Visual Library

The PredixDev public GitHub organization stores a Predix UI CSS library of visual modules for the Predix UI platform to ensure consistent design across a web application..

As a developer, you can use the code in the following modules to produce these interactions.

Note: See https://predixdev.github.io/predix-ui/?show=use_css&type=local for more information using the Predix UI CSS visual libraries in the PredixDev public GitHub organization.
Table 1. Predix UI CSS Visual Library
Module Description Documentation
px-actionable-text-icons-design Actionable text applies button style to text string or icon, allowing a user to associate an action with the text. API Reference
px-button-group-design Implements button groups (multi- or unique-select). API Reference
px-buttons-design Extensible baseline module for building suites of buttons. API Reference
px-code-design Styling for code snippets. API Reference
px-meta-buttons-design Shortcut collection of Predix UI button-related modules. API Reference
px-colors-design Assigns SASS variables for the Predix UI default color palette. API Reference
px-forms-design Defines input fields and styles for form elements. API Reference
px-headings-design Defines font sizes for base-level heading elements. API Reference
px-helpers-design Generic utilities such as calculatetoRem and visually hidden. API Reference
px-input-group-design Simple implementation of input groups. API Reference
px-list-bare-design Removes bullets and indentations from lists. API Reference
px-list-inline-design Displays a list as one horizontal row. API Reference
px-list-ui-design Creates blocked, keyline-delimited list items. API Reference
px-meta-lists-design Shortcut collection of all Predix UI list-related modules. API Reference
px-tables-design Provides useful helpers for common <table> patterns. API Reference
px-toggle-design Simple implementation of a toggle switch for on/off states. API Reference
Note: The API reference documentation for each module includes usage instructions and examples.

File a GitHub issue if you encounter a problem using a Predix UI CSS module. For more information on filing a GitHub issue, see https://guides.github.com/features/issues/.
