Displaying a Multi-Card Deck

This example web application illustrates how the View service database is used to control the visual summary of user information in a web application.

  • See View Service Database Setup for information about configuring the View service database to manage card and deck objects.
  • Select a REST client (such as Postman) that can execute API requests to use the View service. See View Service Database Setup for more information.

    For more information about these APIs, see the API Documentation.

  • Set up the required services for the Predix UI platform. See Setting Up the Predix UI Developer Environment for more information.
  • Locate the project directory of your web application <web_app_project> where the following files will be installed:
    • webcomponents-lite polyfill
    • px-cardrepository from the PredixDev public GitHub organization. See the README file for more information.
  1. Change to the project directory of your web application:
    cd <web_app_project>
  2. Install the webcomponents-lite polyfill (to support the Predix UI repository features) by using the bower package-management utility:
    bower install webcomponentsjs --save
  3. Add the webcomponents-lite polyfill to your web application as the first line in the <head> section of the HTML page. For example
    <!DOCTYPE html>

    <html>

        <head>
            <script src="bower_components/webcomponentsjs/webcomponents.js"></script>

        </head>

     </html>
  4. Install the px-card repository from the PredixDev public GitHub organization.
    bower install https://github.com/PredixDev/px-card.git --save
  5. Import components from the px-card repository into your web application by adding them to the <head> section of the HTML page. For example:
    <!DOCTYPE html>

    <html>
    
      <head>

        <script src="bower_components/webcomponentsjs/webcomponents.js"></script>

  
        <link rel="import" href="/bower_components/px-card/px-dashboard.html"/>
        <link rel="import" href="/bower_components/px-card/px-deck.html"/>
        <link rel="import" href="/bower_components/px-card/px-card.html"/>

     </head>

 
  </html>

    The component files are installed in the bower_components directory.

  6. Call the px-card repository in your web application by adding it to the <body> section of the HTML page. For example:
    <!DOCTYPE html>

    <html>
    
      <head>

        <script src="bower_components/webcomponentsjs/webcomponents.js"></script>

  
        <link rel="import" href="/bower_components/px-card/px-dashboard.html"/>
        <link rel="import" href="/bower_components/px-card/px-deck.html"/>
        <link rel="import" href="/bower_components/px-card/px-card.html"/>

     </head>

     <body>

         <px-card></px-card>

     </body>


  </html>
  7. Create card instance objects in View service.
    • Add cards to the <body> section of the HTML page. For example:
      <!DOCTYPE html>

    <html>
    
      <head>

        <script src="bower_components/webcomponentsjs/webcomponents.js"></script>

  
        <link rel="import" href="/bower_components/px-card/px-dashboard.html"/>
        <link rel="import" href="/bower_components/px-card/px-deck.html"/>
        <link rel="import" href="/bower_components/px-card/px-card.html"/>

     </head>

     <body>

         <px-card></px-card>

         <!-- a card that can display temperature of a city -->  
         <my-temperature-card id="card-1" title="San Ramon Temperature"postal="94582" unit="F"></my-temperature-card>
         <my-temperature-card id="card-2" title="Walnut Creek Temperature"postal="94595" unit="F"></my-temperature-card>


     </body>


  </html>
    • In a REST client, enter the following POST request:
      [POST] /api/cards

      Copy the following JSON blocks into the request body:

      [{
    "title": "San Ramon Temperature",
    "slug": "my-temperature-card",
    "attributes": {
        "postal": "94582",
        "unit": "F"
    }
},
{
    "title": "Walnut Creek Temperature",
    "slug": "my-temperature-card",
    "attributes": {
        "postal": "94595",
        "unit": "F"
    }
}]
  8. Create a deck instance object in View service.
    • Add a deck to the <body> section of the HTML page. For example:
      <!DOCTYPE html>

    <html>
    
      <head>

        <script src="bower_components/webcomponentsjs/webcomponents.js"></script>

  
        <link rel="import" href="/bower_components/px-card/px-dashboard.html"/>
        <link rel="import" href="/bower_components/px-card/px-deck.html"/>
        <link rel="import" href="/bower_components/px-card/px-card.html"/>

     </head>

     <body>

         <px-card></px-card>

         <!-- a card that can display temperature of a city -->  
         <my-temperature-card id="card-1" title="San Ramon Temperature"postal="94582" unit="F"></my-temperature-card>
         <my-temperature-card id="card-2" title="San Ramon Temperature"postal="94582" unit="F"></my-temperature-card>

         <!-- a deck with 2 cards -->
         <px-deck id="deck-1">
             <my-temperature-card id="card-1" title="San Ramon Temperature"postal="94582" unit="F"></my-temperature-card>
             <my-temperature-card id="card-2" title="Walnut Creek Temperature" postal="94595" unit="F"></my-temperature-card>
         </px-deck>

     </body>


  </html>
    • In a REST client, enter the following POST request:
      [POST] /api/decks

      Copy the following JSON block into the request body:

      {
    "title": "Contra Costa Deck"
}
  9. In a REST client, associate card 1 and 2 with deck 1.
    1. Enter the following POST request:
      [POST] /api/decks/1/cards/add
    2. Copy the following JSON block into the request body:
      ["1", "2"]
  10. In a REST client, retrieve all cards in deck 1:
    1. Enter the following GET request:
      [GET] /api/decks/1/cards
    2. Copy the following JSON blocks into the request body:
      {
    "id": 1,
    "title": "Contra Costa Deck",
    "cards":[
        {
            "id": 1,
            "title": "San Ramon Temperature",
            "slug": "my-temperature-card",
            "attributes": {
                "postal": "94582",
                "unit": "F"
            }
        },
        {
            "id": 2,
            "title": "Walnut Creek Temperature",
            "slug": "my-temperature-card",
            "attributes": {
                "postal": "94595",
                "unit": "F"
            }
        }
    ]
}
  11. In a REST client, tag deck 1 to group by categories.
    1. Enter the following POST request:
      [POST] /api/decks/1/tags
    2. Copy the following JSON blocks into the request body:
      [
{"value": "contra costa"},
{"value": "california"},
{"value": "us"}
]
  12. In a REST client, search and retrieve a deck by tag/category.
    1. Enter the following GET request:
      [GET] /api/decks/tags?values=us
    2. Copy the following JSON blocks into the request body:
      [
    {
        "id": 1,
        "title": "Contra Costa Deck"
        "tags": [
            {"value": "contra costa"},
            {"value": "california"},
            {"value": "us"}
        ]
    }
]
  13. Add the px-dashboard component to retrieve all cards in a deck stored in View service by adding it to the <body> section of the HTML page of your web application. This component can traverse the JSON blocks, create DOM objects and append them to an web application. For example: 
    <!DOCTYPE html>

    <html>
    
      <head>

        <script src="bower_components/webcomponentsjs/webcomponents.js"></script>

  
        <link rel="import" href="/bower_components/px-card/px-dashboard.html"/>
        <link rel="import" href="/bower_components/px-card/px-deck.html"/>
        <link rel="import" href="/bower_components/px-card/px-card.html"/>

     </head>

     <body>

         <px-card></px-card>

         <!-- a card that can display temperature of a city -->  
         <my-temperature-card id="card-1" title="San Ramon Temperature"postal="94582" unit="F"></my-temperature-card>

         <!-- a deck with 2 cards -->
         <px-deck id="deck-1">
             <my-temperature-card id="card-1" title="San Ramon Temperature"postal="94582" unit="F"></my-temperature-card>
             <my-temperature-card id="card-3" title="Walnut Creek Temperature" postal="94595" unit="F"></my-temperature-card>
         </px-deck>

         <!-- web component to add to the page -->
             <px-dashboard view-service-base-url=“/api/view-service" selected-deck-id=“1"></px-dashboard>
 
         <!-- Web component will read the json, convert into dom element and inject into the document
              <px-dashboard selected-deck="/api/decks/1?filter[include][cards]">
                  <container>
                      <px-deck id="deck-1">
                          <my-temperature-card id="card-1" title="San Ramon Temperature"postal="94582" unit="F"></my-temperature-card>
                          <my-temperature-card id="card-3" title="Walnut Creek Temperature" postal="94595" unit="F"></my-temperature-card>
                      </px-deck>
                  </container>           
             <px-dashboard>
 
    </body>


  </html>
