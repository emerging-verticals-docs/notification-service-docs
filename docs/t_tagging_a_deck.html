Tagging a Deck

Procedure for tagging a deck or card in the View service database

Select a REST client (such as Postman) that can execute API requests to use the View service.

For more information about this API, see the API Documentation.

You can tag a card or deck for look up by category in the View service database.
  1. Tag a deck to group by multiple values. For example:
    1. Enter the following POST request:
      [POST] /api/decks/1/tags
    2. Copy the following JSON blocks into the request body:
      [
{"value": "contra costa"},
{"value": "california"},
{"value": "us"}
]
  2. Look up a deck by tag. For example:
    1. Enter the following GET request:
      [GET] /api/decks/tags?values=us
    2. Copy the following JSON blocks into the request body:
      [
    {
        "id": 1,
        "title": "Contra Costa Deck"
        "tags": [
            {"value": "contra costa"},
            {"value": "california"},
            {"value": "us"}
        ]
    }
]
