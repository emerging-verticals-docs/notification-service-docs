Predix UI Components Repositories

Predix UI Components descriptions

Predix UI includes a set of components repositories in the PredixDev public GitHub organization that are designed to achieve more complex user interactions than with Predix UI Basics alone when creating the user interface of a web application.

As a developer, you can use the code in the following Predix UI components repositories to produce these interactions.

Table 1. Predix UI Components Repositories
Repository Description Documentation
px-alert-message A user interface component that displays alert information to a user. API Reference
px-app-nav A user interface component that displays a navigation bar or drawer for applications. API Reference
px-calendar-picker A user interface component that allows the user to select a date or date range. API Reference
px-chart A data visualization component that defines a chart, using sub-elements for axis, series and control configuration. API Reference
px-chart-series-bar A data visualization component that defines a chart series that is displayed as a bar. API Reference
px-chart-series-histogram A data visualization component that defines a chart series that is displayed as a histogram. API Reference
px-chart-series-line Adata visualization component that defines a chart series that is displayed as a line. API Reference
px-chart-series-scatter A data visualization component that defines a chart series that is displayed as a scatter. API Reference
px-chart-yaxis A data vissualization omponent used for y-axis configuration for a chart. API Reference
px-clipboard A user interface component that copies a value into the user's computer API Reference
px-context-browser A user interface content browser displaying multiple levels of hierarchy for the current item. API Reference
px-data-table A user interface component that displays a data table with the expected data format. The data is passed as a JSON array. API Reference
px-datetime-field A user interface component that allows the user to select a date and time precisely to the second or millisecond. API Reference
px-datetime-picker A user interface component that allows the user to select a date and time precisely to the second or millisecond. API Reference
px-datetime-range-field A user interface component that allows the user to select a date and time range precisely to the second or millisecond. API Reference
px-datetime-range-panel A user interface component that allows the user to select a date and time range precisely to the second or millisecond. The main use of this component is to be used in the px-rangepicker. API Reference
px-deck-selector A user interface component to draw a drop-down display of available decks that returns the URL for a selected deck. API Reference
px-dropdown A user interface component that displays a customizable dropdown list. API Reference
px-modal A user interface component that displays user interaction workflow using a modal. API Reference
px-overlay A user interface component that displays an overlay with other user interface components. API Reference
px-percent-circle A data visulaization component to draw the simplest possible doughnut chart - a single ring that fills clockwise based on a percentage. API Reference
px-polymer-font-awesome An element providing a starting point for your own reusable Polymer elements. API Reference

px-simple-bar-chart A data visualization component that displays a series of numeric values as a bar chart, or multiple series of numeric values as a stacked bar chart. API Reference

px-simple-line-chart A data visualization component that displays a series of numeric values as a line chart. API Reference

px-vis-parallel-coordinates A data visualization component to draw a parallel coordinates chart API Reference
px-vis-pie-chart A data visualization component to draw a pie or donut chart for some data. API Reference
px-vis-polar A data visualization component to draw a phase and amplitude on a polar plot. API Reference
px-vis-radar A data visualization component to draw a radar (spider) chart. API Reference

px-vis-timeseries A data visualization component to create an interactive timeseries chart. API Reference
px-vis-xy-chart A data visualization component to draw lines, scatter or both against an X and Y axis. API Reference
Note:

The API reference documentation for individual repositories also includes usage instructions and examples.

File a GitHub issue if you encounter a problem using a Predix UI repository. For more information on filing a GitHub issue, see https://guides.github.com/features/issues/.
