Installing the Predix UI CSS Starter Kit

The px-starter-kit-design library serves as a starting point for a new Predix UI CSS project.

Set up the required environment for Predix UI. For more information, see Setting Up the Predix UI Developer Environment.
Note: See https://predixdev.github.io/predix-ui/?show=px-starter-kit-design&type=css for more information on installing the Predix UI CSS files stored on the PredixDev public GitHub organization..
  1. Change to the application project folder:
    cd <app_project>
  2. Install the Predix UI CSS starter kit files: 
    bower --save install https://github.com/predixdev/px-starter-kit-design.git

    The Predix UI CSS starter kit files are installed in the bower_components directory.

  3. Install the grunt task-runner utility using npm:
    npm install grunt -g
  4. Install the import once module: using npm:
    npm install node-sass-import-once --save-dev
  5. Install the autoprefixer utility using npm that parses CSS and adds vendor prefixes to CSS rules:
    npm install grunt-autoprefixer --save-dev
