Predix UI Mobile Components Repositories

Predix UI Mobile Components descriptions

Predix UI includes a set of components in the PredixDev public GitHub organization that provide a foundation for building web applications to look and work like native mobile applications:.

As a developer, you can use the code in the following Predix UI mobile component repositories to produce these interactions.

Table 1. Predix UI Mobile Components Repositories
Repository Description Documentation
px-drawer A user interface component that provides an off-screen menu drawer that can be swiped open or closed. API Reference
px-table-view A user interface component that helps organize menus and views into table-style blocks that can be expanded, swiped, and dragged and re-ordered. API Reference
Note:

The API reference documentation for individual repositories also includes usage instructions and examples.

File a GitHub issue if you encounter a problem using a Predix UI repository. For more information on filing a GitHub issue, see https://guides.github.com/features/issues/.
